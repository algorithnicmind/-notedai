## 🌟 Overview
**Noted** is a sleek, multi-user web notebook that runs entirely in your browser. With a state-of-the-art dark mode UI featuring **glassmorphism**, **smooth micro-animations**, and **real-time filtering**, it's the fastest way to jot down ideas safely.

### 🎨 Key Features
- **🚀 Ultra-fast Performance**: Zero-latency typing with zero external network requests.
- **🔐 User Authentication**: Create an account or sign into an existing one. Your notes are securely tied to your profile.
- **💾 Auto-Save**: Never lose a thought. Every keystroke is saved effortlessly with a smart 500ms debounce.
- **💅 Premium Aesthetics**: Vibrant gradients, carefully curated `Inter` typography, and stunning hover states.
- **🔍 Instant Search**: Filter your entire notebook seamlessly.

## 🛠 Getting Started

Taking notes is just a double-click away since **Noted** doesn't require any complex build steps!

### 1. Launching the App
Simply open the root `index.html` file in any modern web browser:
```bash
# Windows
Start-Process index.html

# MacOS
open index.html
```
*(Alternatively, you can just manually open the index.html file with your favorite browser!).*

### 2. Signing In
Upon your first visit, you will be beautifully redirected to the authentication gateway. 
- Click **Sign Up** to create your local persistent account.
- Already have an account? Just type your credentials and step right back into your notebook.
